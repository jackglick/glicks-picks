Methodology

Built on pitch-level data

Every feature is engineered from raw Statcast pitch data — no pre-aggregated stats.

☰

Data

Statcast pitch-level data from 2022–2024 for training. Full 2025 season held out for unbiased validation. ~8,800 starter games with minimum 5 prior starts.

⚙

Features

14 engineered features: rolling K rates, lineup composition, park factors, umpire tendencies, pitch velocity range, swinging-strike rate, fatigue modeling, and more.

⚡

Model

Poisson GLM blended with XGBoost (60/40). Poisson CDF converts predictions to probabilities for computing betting edge against market-implied odds.

Betting Edge

Finding value in the market

When the model disagrees with the sportsbook, that gap is the edge.

Edge = Model Probability − Implied Market Probability

The model predicts a strikeout total using Poisson regression, then computes the probability of going over or under a given line. When that probability exceeds the market's implied probability (derived from the odds), we have an edge.
