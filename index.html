How it works

Four years of pitch-level Statcast data, distilled into outcome probabilities for each market.

01

Build features

Every pitch from 2022–2025 — velocity, movement, location, outcome — plus context: park factors, home-plate umpire tendencies, batter/pitcher platoon splits, and rolling recent form.

02

Predict outcomes

ML ensembles estimate the probability distribution for each stat line — strikeouts, earned runs, outs, hits, walks, batter hits, and total bases.

03

Compare to odds

Model probabilities are compared to implied odds from sportsbook lines. Bets are included only when the difference exceeds a per-market threshold (10–15%).

Why player props?

In our testing, game-level markets (totals, moneyline, run lines) showed no exploitable signal. Player props showed larger and more persistent differences between model probabilities and implied odds, particularly in batter and pitcher counting stats.

About this project

Built by Jack Glick, a data scientist and lifelong baseball fan. This started as a personal project to test whether machine learning could find meaningful signal in MLB player props, then expanded into a seven-market system covering pitcher and batter outcomes.

This is a research project, not a picks service. All statistics shown on this site are from backtested, out-of-sample 2025 archive data — not live wagering.
